# Amazon Automation Assignment

## Overview
This repository contains automated test cases for Amazon product search and cart functionality.

## Tech Stack
- Language: JavaScript
- Framework: Playwright

## Test Cases
1. Search for an iPhone, add to cart, and retrieve price
2. Search for a Galaxy device, add to cart, and retrieve price

## Parallel Execution
Playwright configuration is set to execute test cases in parallel using multiple workers.

## Cloud Execution
The framework supports execution on LambdaTest using remote browser capabilities.
